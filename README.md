# 온체인 레이싱 봇 — 기술 노트

> Paradigm CTF 2022 (0xMonaco) 출품작 코드를 기반으로, 해당 도메인의 일반적인 패턴과 트레이드오프를 정리한 관찰자 시점 문서. 챌린지 자체는 공개되어 있으며, 이 문서는 **경매 기반 온체인 레이싱 게임에서 동작하는 AI 봇의 구조와 통념**에 초점을 둔다.

**카테고리:** 동적 가격 경매가 결합된 온체인 레이싱 게임용 AI 전략 봇
**스택:** Solidity 0.8.13, Foundry, Solmate, fixed-point math (SignedWadMath)
**눈여겨볼 지점:**
- 양방향 시장 (`ACCELERATE` vs `SHELL`) 이 **VRGDA** 스타일의 연속 시간 감쇠 곡선으로 가격이 매겨짐 → 봇은 온체인에서 계산 가능한 곡선을 상대로 매수 타이밍을 잡는 구조
- 봇 로직 전체가 `takeYourTurn(carsState[], myIndex)` 한 함수 안에 들어감 — 영구 저장소 없이, 매 턴 순수 상태 머신처럼 추론하는 형태
- 적대적 환경: 매 게임 두 명의 다른 봇이 동시에 존재 → 단순 수식 풀이보다 **상대 모델링이 더 결정적인 변수**가 되는 영역

## 이 문서를 읽으면 좋은 사람

다음 조건의 온체인 에이전트를 만들고 있다면 동일한 패턴이 적용된다:

- **연속 가격 경매**를 추론해야 하는 경우 (`x*` 토큰 세일, VRGDA로 가격이 매겨지는 NFT mint, 가스 제한 리소스 마켓 등)
- **가스 예산**과 **고정된 컨트랙트 API** 안에서 매 턴 액션을 결정해야 하는 경우
- 온체인 텔레메트리로만 행동을 추론할 수 있는 **다른 에이전트들과 경쟁**해야 하는 경우
